A browser-based quiz board interface built with JavaScript, HTML, and CSS. The project focuses on dynamic UI rendering, event-driven interactions, and client-side state management.
Overview
This application simulates a category-based quiz board using dynamic DOM updates. Users can select questions from a grid interface, triggering content reveal and state transitions.
The goal of the project was to practice structured front-end development and interactive UI design without relying on frameworks.
Core Technical Features
Dynamic DOM generation
Event delegation and click handling
State-based content updates
Grid layout design
Responsive styling with CSS
Modular script organization
